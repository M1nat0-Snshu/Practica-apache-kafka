# Practica-apache-kafka
Esta es una práctica en la cual hacemos una microarquitectura de transmisión de datos utilizando apache kafka, zookeeper, contenedores de docker y python

La estructura necesaria para este proyecto es la siguiente:

practica

├── kafka_consumer.py

├── kafka_producer.py

├── kafka-single-node

│   └── docker-compose.yml

└── venv

    ├── bin
    ├── include
    ├── lib
    ├── lib64 -> lib
    └── pyvenv.cfg

requiere instalar kafka-python desde pip
